# The Mayhem Tour – Ticketmaster Mobile UI Clone

Frontend clone of the Ticketmaster mobile experience, built entirely in a single HTML file containing markup, styling, and JavaScript.

Live demo: [https://the-mayhem-tour.netlify.app](https://the-mayhem-tour.netlify.app)

## Overview

Recreates key mobile flows:

* Event browsing
* Tour detail view
* Mobile-first layout
* Interactive UI with vanilla JavaScript

## Tech Stack

* HTML
* CSS
* Vanilla JavaScript
* Netlify deployment

No frameworks or build tools.

## Architecture

Everything lives in one `.html` file:

* `<style>` for CSS
* `<script>` for JS
* Structured sections for UI components

Built to practice layout precision, UI replication, and clean organization under constraints.
